### LINKED LIST

#### Characteristics of Linked List.
- Dynamic: (Động) Linked list triển khai các phép toán thêm/xóa với độ phức tạp O(1).
- Non-contiguous: (Không liền kề) Linked List lưu các node ở các địa chỉ ngẫu nhiên trên địa chỉ thanh ram và connected với nhau thông qua pointer.
- Sequential access: (Truy cập tuần tự): Các node được thăm một cách tuần tự thông qua node head.
#### Types of Linked List.
- Singly Linked List.  
- Doubly Linked List.  
- Circular Linked List.
#### Linked List Operation.
- Creation.
- Traversal.
- Insertion.
- Deletion.
- Search.
